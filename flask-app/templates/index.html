Drone Image Detection Dashboard

Use of various drones to capture aerial images and identify the number of people in that image. Coupled with real time GPS sensing, this webiste provides a list of the exact location and the number of people detected in that area. Useful for deployment during any emergency or crisis.

Implementation

The pipeline uses a Raspberry Pi to send images captured by a drone to Google Drive. Google Colab allows us to use a GPU to process the images and detect people. The images are streamed to a Flask webpage, as well as the corresponding GPS coordinates for the images.
